Il est peut-être nécessaire de supprimer mono.android qui est dans le projet Fourplaces

Si Mono.Android est présent dans les dépendances de Fourplaces, il est nécessaire de le supprimer, sinon la compilation ne fonctionne pas . Il est peut-être aussi nécessaire de mettre à jour les références de Fourplaces.Android.


Il est aussi possible d'installer l'apk de l'app. Mais d'après ce que j'ai vu elle ne fonctionne qu'une seule fois sur son smartphone(on ne peut pas la réinstaller si on l'a désinstallé une fois).Et lorsqu'elle s'installe il faut cliquer sur "installer quand même" sinon l'application ne s'installera pas.
